# 代数极：GF(3)/C3/A4 共轭引擎

> **代数极是频率域的动力学引擎**。突破 GF(2) 的维度锁死，注入 GF(3)→GF(3²)→C3→A₄ 的共轭链。

## GF(2) 的盲区

- GF(2) 的伽罗瓦群是 Z₂ — 只能产生平凡二阶自同构 (正负号翻转)
- 实数域 R 的自同构群是平凡的 — 无法自动感知 √3 的共轭 -√3
- **后果**: 手征性 (左旋/右旋) 只能作为外部补丁引入 (如 CP 破坏相角)

## GF(3)→GF(3²)→C3 共轭链

### GF(3) 加法底座

```
GF3 = {0, 1, 2}
T6Lattice = Vec GF3 6    — 729 格点
```

GF(3) 提供离散演化的纯粹空间骨架 (平移底座)。

### GF(3²) 乘法纤维 — 代数共轭的自动生成器

多项式 x² + x + 1 在 GF(3) 上不可约。
真正的三次单位根 ω 存在于 GF(3²) (9个元素):

```
GF(3²) = GF(3)[x]/(x² + x + 1)
ω  = x mod (x²+x+1)
ω² = 2x+2 mod (x²+x+1)
Frobenius: x ↦ x³ ≡ 2x (mod x²+x+1) — 即 ω ↦ ω²
```

**关键**: ω 和 ω² 是天然的伽罗瓦共轭对，通过 Frobenius 自同构完美互换。
这种共轭是域扩张内部自动涌现的 — 无需外部补丁。

### C3 循环群

C3 是三阶循环群，对应 ω 相位的 120° 旋转。
在 T⁶ 晶格上，C3 作用产生驻波干涉的三拍周期。

## A₄ 群 — 代数与几何的桥梁

A₄ 是正四面体旋转对称群, 12 个元素:
- 1 个恒等元
- 8 个 3 阶旋转 (C3 生成元)
- 3 个 2 阶翻转

在律算框架中，A₄ 将代数共轭 (C3) 直接绑定到三维空间的几何对称性上。
**伽罗瓦群 (纯代数) 与基本群 (纯几何) 在此重合。**

在当前框架中，CRT 不是模运算双射，而是**双振子拍频干涉的谐波谱投影**:

```
CRT 投影: Z/(pq) → Z/p × Z/q
        = 一维线性时间 → 二维正交频率域
```
