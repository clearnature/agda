# 几何极：T⁶ 晶格与幻方正交

> **几何极是代数动力学的物理载体**。计算状态不是标量，而是 T⁶ 晶格上的位置向量。

## T⁶ 量子晶格

```
GF3 = Fin 3          — GF(3) 域
T6Lattice = Vec GF3 6 — 3⁶ = 729 个离散格点
```

- **维度**: 实 6 维环面 (Torus)
- **格点数**: 729 (保留粒子演化的完整空间位置表征)
- **底座**: GF(3) 加法群 — 平移对称性
- **纤维**: GF(3²) 乘法群 — 旋转对称性 (手征)

## 亚瑟幻方正交场

### 幻方的组合结构

4×4 幻方：每行、每列、对角线之和相等 (幻和 = 34)。

在编译器内核中，幻方正交约束替代了一维的 Path 拼接:
- **行列权重守恒**: 独立于单一路径
- **组合正交性**: 行向量与列向量在权重空间中垂直
- **刚性网格骨架**: 确保空间形变下总能量/维度平衡

### 谱截断定理 (引理 1)

> 设 λ_R ∈ ℝ 为 Serre 四阶幻方 M₄ 的无理本征值。
> 则 λ_R 在 CRT 坐标系 ∏Z_{m_i} 中无有效表示。
> 令 P_CRT: ℝ → ∏Z_{m_i} 为 CRT 域投影映射:
> P_CRT(λ_R) ≡ λ_CRT
> 其中 λ_CRT 为最近邻整数同余类标签。
> 对于 λ_R = ±2√10，投影截断为 λ_CRT = ±16。

**物理意义**: 无理本征值 ±2√10 在连续统中存在，但在离散 CRT 格点上被截断为 ±16。这就是编译器手征分裂 (±16) 的深层几何来源。

### 正交判据替换 (引理 2)

> 设 m_i, m_j ∈ CRT 模数集合。
> 正交性判据由幻方本征向量在离散格点上的拓扑内积定义:
> ⟨v_{λ_i}, v_{λ_j}⟩_{Z_144×Z_46} = 0
> 此判据替换经典 CRT 的 gcd(m_i, m_j) = 1。

**物理意义**: 互质条件在幻方正交框架中被拓扑内积替换。
gcd(144,46)=2 不再是障碍 — 144 和 46 在拓扑格点上可以正交。

## Christoffel 螺旋

Christoffel 螺旋是 T⁶ 环面上的离散测地线 — 不是 2D 曲线:
- **驻波节点** {0,3,6,9}: 空间剖分点 (极向)
- **巡游路径** {1,2,4,8,7,5}: 时域传播相 (环向)
- **相位对齐点** 6624: 极向和环向在此同步 (非闭合!)

在训练框架中，Christoffel 平行移动替代了传统的 Q·K^T Attention:


## 离散测地线与 6624 环面巡游

- 轨迹不是平面曲线，是 T⁶ 环面上的离散格点跃迁
- 驻波节点: 空间剖分 → 极向周期 = 144
- 巡游路径: 时域传播 → 环向周期 = 46
- 6624 格点: 极向 144 和环向 46 的相位对齐 (LCM)
- 对齐 ≠ 闭合: 系统在此同步但不停止，持续级联
