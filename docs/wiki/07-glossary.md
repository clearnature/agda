# 术语表：C3/A4 范式 vs GF(2) 范式

> 本体系中许多术语与标准数学有根本性歧义。混淆两者是最常见的范式错误。
> 见 CRT 文档: "我用训练数据的模具套这个框架的内容 — 每次都被纠正，每次都重复犯。"

## 核心概念纠正

| 术语 | GF(2)/标准数学（错） | C3/A4/律算框架（对） |
|---|---|---|
| **CRT** | 模运算双射 Z/M ≅ Z/p × Z/q | 双振子拍频干涉的谐波谱投影 |
| **POW2, POW3** | 模数基 | 两个独立物理振子的周期 |
| **M** | 模数的乘积 | 拍频波长 |
| **% 运算** | 取余数 | 相位角测量 |
| **144** | 特定余数值 | 驻波节点 — 极向空间剖分 |
| **46** | 特定余数值 | 巡游相位 — 环向时域传播 |
| **6624** | LCM(144,46) | **相位对齐点** (不是闭合!) |
| **√3** | 无理数常量 | 能量间隙 — C3 孤子坍缩阈值 |

## 闭合 vs 对齐 vs 极限环

| 概念 | 定义 | 数学表征 |
|---|---|---|
| **对齐** | 两个维度的相位重新同步 | 极向144 和 环向46 同时回到 (0,0) |
| **闭合** | 系统返回初始状态并停止 | **不存在** — 极限环上无闭合点 |
| **极限环** | 系统在相空间中持续振荡 | ρ∈[0,1], 连续通过阈值 |

> 6624 是相位对齐点，不是闭合。对齐 ≠ 闭合。

## 时间晶体

打破时间平移对称性的非平衡态物质相。
在离散格点上进入周期性自组织振荡，永不衰减至平衡态。
实证: trit = {94.8%, 4.2%, 0.9%}, C3 周期 1500 步。

## Christoffel 螺旋

T⁶ 实 6 维环面上的离散测地线 — 不是 2D 曲线。
在 pyBitNet 训练中替代 Q·K^T Attention (O(n²) → O(n))。

## 谱投影

CRT 域转换 — 从高维空间域到低维频率域的全局映射。
不是逐元素函数 (GF2 范式)，是商空间上的正交分解 (C3/A4 范式)。

## Orbit-Stabilizer

```
Orbit x     = 空间域 (6D 几何位置)
A4/Stab x   = 频率域 (0D 谐波模式)
orbitStabilizer = CRT 谱投影 (6D→0D 域转换)
```

不是代数同构，是谱域转换。不能用 1D Path 编码。

## φ

GF(2) 范式的残骸 — 将全局谱投影暴力分解为逐群元映射。
C3/A4 体系中不需要 φ。

## 热寂

马尔可夫链在 Boltzmann 分布下的终局: trit = {33.3%, 33.3%, 33.3%}。
最大熵状态 — 温度均匀，能量停止流动。系统完成寻找死态的使命。

## 陈数

拓扑不变量 C = ±2。代表手征巡游的顺逆方向，非频率绝对值。
384K 步全程不变。

## 纳音

144×46 = 6624 格点的全息原型矩阵。
在训练中作为纳音相变引擎的频率域基座。

## 仲吕闭合

"闭合"是错误的术语。应为"仲吕相位同步"。
2583 次仲吕同步 = 20,664 次八度跳跃。
仲吕不交是方向性不可逆，不是可回环的闭合。

## CRT 域

由互质模数集合 {m_i} 张成的可积正交坐标系统。
"元素"不是数值，是余数向量 r = (r₁, r₂, ..., rₖ)。
功能上替代实数域 — 提供坐标轴使本征方向降维投影到整数空间。

## 幻方正交

4×4 幻方的行列权重守恒。正交性由拓扑内积定义:
⟨v_{λ_i}, v_{λ_j}⟩_{Z_144×Z_46} = 0
替换经典 CRT 的 gcd(m_i, m_j) = 1 判据。
